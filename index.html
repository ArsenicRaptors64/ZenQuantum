NEMESIS AI is a java based program (still under works) aimed at providing a secure and a private AI system.

PROGRAM STATUS: ALPHA
LATEST BUILD TYPE: PROTOTYPE

The AI has the following features implemented:

  • SecureBoot and FUSE programming (Fractal User Security Engine).
  • Fractal File Saving System.
  • Official Windows™ and Unix based systems supported.
  • Nion Messenger.
  • OTA program updation.
  • Administrator and Standard user privilige tiers.
There are many more features and modules which are coming up. They will be released and the client can download it as per their wish.
